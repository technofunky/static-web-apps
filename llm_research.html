🔁 Integrated pipeline: post‑training → Agentic RL
🧩 Key synergy Post‑training creates robust, aligned LLMs; Agentic RL extends them to dynamic, partially observable worlds.

🔄 Synergy

SFT + RL (DPO/GRPO) → reliable alignment. Agentic RL adds tool use, memory, planning, multi‑turn interaction.

⚙️ Common algorithms

GRPO and PPO bridge both worlds: GRPO for efficiency (DeepSeek‑R1), PPO for general alignment.

🧪 Real‑world agents

OpenAI Deep Research, Kimi K2, WebAgent‑R1, Voyager, SWE agents use RL‑optimised planning + tool execution.

📈 Future trajectory

Hybrid systems: SFT for grounding, RL for reasoning, test‑time scaling for hard queries, agentic frameworks for long‑horizon tasks.

📖 First‑Principles Glossary

LLM (Large Language Model)
A neural network trained to predict the next token in a sequence using maximum likelihood estimation on large text corpora. It learns a probability distribution over language.
MLE (Maximum Likelihood Estimation)
Parameter estimation that maximizes the probability of observed data. In LLMs, MLE minimizes negative log‑likelihood: -Σ log P(token | context).
MDP (Markov Decision Process)
A mathematical framework (S, A, P, R) for sequential decision making where next state depends only on current state and action.
POMDP (Partially Observable MDP)
Extension of MDP where the agent does not directly observe the full state but receives observations O(s). Essential for agentic RL.
Policy π(a|s)
Probability distribution over actions given a state. In LLMs, policy outputs next‑token probabilities.
Reward Model R(x,y)
A function that scores (prompt, response) pairs based on human preferences, trained via Bradley‑Terry or Plackett‑Luce loss.
PPO (Proximal Policy Optimization)
RL algorithm that clips policy updates to prevent destructive changes, using a critic network for advantage estimation and KL penalty.
DPO (Direct Preference Optimization)
Eliminates reward model by reparameterizing preference learning as a classification loss on log‑probability ratios.
GRPO (Group Relative Policy Optimization)
Removes critic network; estimates advantage using group statistics (mean/std) of sampled responses. Used in DeepSeek‑R1.
Chain‑of‑Thought (CoT)
Prompting method that forces step‑by‑step reasoning before final answer, improving performance on multi‑step tasks.
Test‑Time Scaling (TTC)
Spending extra compute during inference (e.g., multiple samples, tree search) to improve accuracy without retraining.
Agentic RL
Reinforcement learning paradigm where LLM acts as autonomous agent in POMDP with tool use, memory, planning, and multi‑turn interaction.
Temporal Credit Assignment
Identifying which actions in a long sequence caused eventual success or failure. Major challenge in multi‑turn agentic tasks.
Outcome Reward Model (ORM)
Rewards only the final answer. Simple but cannot credit intermediate steps.
Process Reward Model (PRM)
Rewards each reasoning step. Better for math/code but expensive to annotate.
SFT (Supervised Fine‑Tuning)
Training on human‑curated (prompt, response) pairs to align model output format and style.
